Adding the required JavaScript snippet for the lemniscus appointment widget into your page or post by using the simple Shortcode [lemniscus]

== Description ==
Include your Lemniscus online appointment scheduling calendar in any page or article.
This plugin requires a lemniscus.de account.
WP Lemniscus is only available in german.

German description:

Lemniscus ist eine moderne webbasierte Praxisverwaltung, die speziell für den selbständigen Heilpraktiker entwickelt wurde. Der erfrischende, ganzheitliche Ansatz nutzt die Synergiepotentiale aus der nahtlosen Integration von Terminplan, Patientenakte und Rechnungsstellung. 

Dank Lemniscus muss sich der Therapeut nicht mehr um die Installation, Wartung, Updates und Datensicherung der Praxissoftware bemühen - er benötigt lediglich einen Internetzugang. 

Lemniscus ermöglicht die Anbindung eines Telefondienstes einer Online-Terminvergabe für die direkte Terminierung durch den Patienten und natürlich eine bemerkenswert einfache Anbindung zu einer Abrechnungsstelle.

Dank Lemniscus können sich auch kleinere Praxen eine professionelle, sichere und datenschutzbewusste Praxisverwaltung leisten. 

Mit diesem Plugin bieten wir eine einfache Einbindung der Lemniscus online Terminvergabe auf beliebige Seiten oder Posts in WordPress. 

== Installation ==
Install plugin
Activate plugin
Enter Token provided by lemniscus.de in settings. Make sure token is activated.
Add the shortcode [lemnsicus] to any page or article
